# code-wiki:schema

管理维基的行为规范。

## 用法

- `/wiki schema show` - 显示当前规范
- `/wiki schema edit` - 编辑规范（在编辑器中打开）

## 前置条件

维基已初始化。

## 命令

### show
显示当前 schema/CLAUDE.md 内容。

### edit
在编辑器中打开 schema/CLAUDE.md 进行修改。
保存后确认规范有效。

## 规范位置

`<维基路径>/schema/CLAUDE.md`
