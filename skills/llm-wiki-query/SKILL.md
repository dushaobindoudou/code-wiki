# llm-wiki:query

查询维基并生成答案。

## 用法

`/wiki query <问题>`

或直接提问 - skill 会自动识别查询意图。

## 前置条件

维基已有内容（至少已摄取一篇文档）。

## 流程

1. **查找相关页面**
   - 读取 index.md 定位相关章节
   - 识别哪些实体/概念/摘要与问题相关

2. **读取相关内容**
   - 加载引用的页面
   - 综合信息

3. **生成答案**
   - 用 Markdown 回答
   - 包含引用：[[页面名]]
   - 询问是否将探索结果存回维基

4. **可选：存回维基**
   - 如果用户愿意，在适当目录创建新页面
   - 更新 index.md
   - 记录到 log.md

## 输出格式

- Markdown（默认）
- 对比表（使用 `||` 语法）
- 摘要列表

## 响应规则

- 必须引用相关页面 [[页面名]]
- 鼓励将有用的探索结果存回维基
- 如果维基为空，提示先摄取文档
